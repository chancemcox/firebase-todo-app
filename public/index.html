Architecture

Frontend: Firebase Hosting (this page)

Backend: Laravel PHP at todo.cox-fam.com

Database: Firebase Realtime Database

Authentication: Laravel Passport with Firebase storage

API Endpoints

Base URL: https://todo.cox-fam.com/api

Authentication: POST /auth/register, POST /auth/login

Todos: GET, POST, PUT, DELETE /todos

User: GET, PUT /user/profile

Local Development

To run Laravel locally:

  1. php artisan serve (runs on http://localhost:8000)
  2. Update .env with your Firebase credentials
  3. Access API at http://localhost:8000/api/*
